# Supply Chain Analytics using SQL

## Project Overview

This project analyzes supply chain data using MySQL to evaluate supplier performance, inventory management, logistics efficiency, and revenue generation.

## Tools Used

* MySQL
* SQL
* MySQL Workbench
* GitHub

## Dataset Information

The dataset contains information about:

* Product Type
* Revenue Generated
* Supplier Name
* Stock Levels
* Lead Time
* Transportation Modes
* Manufacturing Costs
* Defect Rates
* Shipping Costs

## Business Questions Answered

1. Which product category generates the highest revenue?
2. Which suppliers contribute the most revenue?
3. What is the average lead time?
4. Which transportation modes generate the highest revenue?
5. Which product categories maintain the highest inventory levels?

## Key Insights

* Skincare products generated the highest revenue.
* Supplier 1 generated the highest revenue among suppliers.
* Rail transportation contributed the largest revenue share.
* Average lead time was approximately 17 days.
* Inventory levels were relatively balanced across product categories.
